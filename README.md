#  STURBACKS

El siguiente repositorio es una implementación de un ecommerce que toma referencias de una famosa marca y utiliza React para desarrollar los puntos necesarios para la creación de un sitio web.


### Descripción del proyecto

El desarrollo cuenta con un filtro de categorias de bebidas y alimentos, listando unicamente los elementos que pertenezcan a cada categoria y renderiznado cada uno en una página diferentes. 

También cuenta con las interacciones que permiten al usuario agregar un elemento al carrito, al igual que la modificación de la cantidad de elementos a comprar durante la selección.

El carrito de compras visualiza la totalidad del valor de la cantidad por elementos iguales y permite descartar la compra por elementor diferentes o todos a la vez.

La finalización de la compra genera un formulario de registro que valida el nombre del usuario, correo electronico y número de telefono, estos datos son recopilados y enviados a una base de datos no relacional.

La base de datos se encuentra implementada en Firebase y contiene el registro de compras junto al listado de productos que se muestran en el ecommerce.

## Instrucciones

Para descargar y ejecutar el proyecto solo debe clonar el repositorio e instalar los módulos correspondientes.

```
npm i
```

Luego se realizara el inicio del servidor local.

```
npm start
```
